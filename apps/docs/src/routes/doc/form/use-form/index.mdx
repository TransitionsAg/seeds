# useForm

Creates a reactive form backed by a single SolidJS store. Use `useFormComponents` to get `Form` and `Field` components wired to the form instance.

## Usage

```tsx
import { useForm, useFormComponents } from "@transitionsag/form";

const form = useForm({
  initialValues: { email: "", password: "" },
  onSubmit: (value) => console.log(value),
});

const { values, errors, state, submit, reset } = form;
const { Form, Field } = useFormComponents(form);
```
