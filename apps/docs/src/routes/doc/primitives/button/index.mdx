# Button

An unstyled button primitive with polymorphic support.

## Installation

```bash
pnpm add @transitionsag/primitives/button
```

## Usage

```tsx
import { Button } from "@transitionsag/primitives/button";

<Button>Click me</Button>
<Button as="a" href="/link">
  As a link
</Button>
```
