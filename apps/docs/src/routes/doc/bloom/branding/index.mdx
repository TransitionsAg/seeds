# Branding

Theme context for runtime color customization. Wrap children in a div with CSS custom properties derived from the theme object.

## Usage

```tsx
import { Branding } from "@transitionsag/bloom";

<Branding
  theme={{
    primary: "#3b82f6",
    "primary-accent": "#2563eb",
    background: "#ffffff",
    foreground: "#111827",
  }}
>
  {children}
</Branding>;
```

## useBranding Hook

```tsx
import { Branding } from "@transitionsag/bloom";

const theme = Branding.useBranding();
```
